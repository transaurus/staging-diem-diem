Query the blockchain

The JSON-RPC service provides APIs for clients to query the Diem Blockchain. This tutorial will guide you through the different methods you can use. You can test these on testnet.

Assumptions#

All commands in this document assume that:

  • You are running on a Linux (Red Hat or Debian-based) or macOS system.
  • You have a stable connection to the internet.

Prerequisites#

  • Complete the My first transaction tutorial to understand how to create accounts and interact with testnet.

If you have already completed the My First Transaction tutorial, you can use the following account information for Alice and Bob to get started:

  • Account address: Hex-coded account address

Get account information#

To get information about Alice’s account, use get_account. To run this method, you will need the hex-coded account address (see step 4 of My First Transaction).

In the example below, we have a request sent using get_account to learn information about Alice’s account and the response received.

  • Alice’s hex-code account address

All methods#

You can use different methods to query the blockchain and get the information you’re looking for. We’ve included the complete list and their descriptions below.

MethodDescription
get_transactionsGet transactions on the Diem Blockchain.
get_accountGet the latest account information for a given account address.
get_account_transactionsGet transactions sent by a particular account.
get_metadataGet the blockchain metadata (for example, state as known to the current full node).
get_eventsGet the events for a given event stream key.
get_currenciesGet information about all currencies supported by the Diem Blockchain.
